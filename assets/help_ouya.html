ConSoul

Play your favourite old DOS games!

Control choices

  1. OUYA Controler
    • O - Enter key
    • U - Space key
    • Y - "y" key
    • A - Escape key
    • L1 - Control key
    • R1 - Alt key
    • R2 - Shift key
    • Left Joystick - arrow keys
    • Dpad - arrow keys
  2. Touchpad with the onscreen keyboard (which you can toggle via the menu).
  3. USB or bluetooth physical keyboard

Press any key to continue...
